Configuration instruction for Windows

1. Install Windows version of node.js
2. Install Windows version of MongoDB
3. Create database http://docs.mongodb.org/manual/tutorial/install-mongodb-on-windows/
4. Install node.js packages:
	- express
	- express body parsers
	- mongoskin
	
	
To run test server
1. Start mongod process
3. start node with main server script
Project contains PowerShell ISE script to start both services in single window
